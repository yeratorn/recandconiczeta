**Test codes for the paper:** Reciprocity For Dedekind Sums via Conical Zeta Values 
**Author:** Yerko Torres-Nova

This repository contains the reference implementation of the methods described in the paper, plus scripts to reproduce the main experiments/figures/tables.

- **Paper:** https://arxiv.org/abs/2512.20542 

## Repository structure

Each file testsc(number).py contains functions that allow testing the statements of the section (number) of the paper.
